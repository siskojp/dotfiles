#Installation

	git clone git://github.com/gcoguiec/dotfiles.git

	cp .gitconfig ~

A little advice for lazy programmers : alias g='git'

	g i
	g o git://...
	g st
	g a .
	g c "initial stuff."
 	g p
	g ll
